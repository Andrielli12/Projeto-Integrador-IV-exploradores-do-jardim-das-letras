# 🌿 Exploradores do Jardim das Letras

**Projeto Integrado IV – UNIVESP**  
*Licenciatura em Pedagogia*

---

## 📖 Sobre o Projeto

O **Exploradores do Jardim das Letras** é um jogo educativo desenvolvido como parte do Projeto Integrado IV da UNIVESP. O objetivo é auxiliar crianças em fase de alfabetização (aproximadamente 7 anos) a reconhecer palavras e associá-las a elementos da natureza, estimulando a leitura, a escrita e a consciência ambiental.

O jogo apresenta um jardim encantado com nove palavras relacionadas ao meio ambiente. A criança deve ler a dica de cada palavra e preencher os quadradinhos com as letras corretas. Ao acertar, ganha um ✅ no card e, ao completar todas, recebe uma medalha especial.

---

## 🎮 Funcionalidades

- Tela inicial com jardim animado (árvores, flores, nuvens, pássaros, borboletas)
- Botão **JOGAR** para acessar os cards
- Botão **RESET** para reiniciar o progresso (nova partida)
- 9 cards interativos, cada um representando uma palavra
- Cada card tem uma cor de borda específica (ex: sol = amarelo, flor = rosa, água = azul)
- Ao clicar no card, abre a tela da palavra com:
  - Cenário ilustrado (emojis usados como placeholder)
  - Texto explicativo (dica)
  - Quadrados para digitar a palavra
  - Botões VERIFICAR, TENTAR NOVAMENTE e VOLTAR
- Feedback visual para acertos (borda verde), erros (borda vermelha) e falta de preenchimento
- Progresso salvo no navegador (mesmo fechando a página, as palavras completadas continuam marcadas)
- Medalha comemorativa ao finalizar todas as palavras

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura das telas e elementos
- **CSS3** – Estilização, animações e design responsivo
- **JavaScript (ES6)** – Lógica do jogo, controle de estado, validação das palavras e armazenamento local
- **Google Fonts** – Fonte infantil (`Comic Neue`) para tornar a leitura mais agradável
- **LocalStorage** – Persistência do progresso do jogador
